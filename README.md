# Project-KYC-Verification-System1. Project: KYC Verification System

📌 Objective

Simulate how banks verify customer identity during onboarding.

🧾 Description

This project checks whether a customer is compliant based on:

PAN validation
Aadhaar format
Missing documents
Risk categorization
🎯 Features
Input customer data
Validate documents
Assign risk level (Low / Medium / High)
Flag incomplete KYC
